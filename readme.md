# Botany Pots Fix 植物盆栽修复

修复/减轻了植物盆栽对服务端与客户端的卡顿。

如果服务端安装，则能减缓server tick time；
如果客户端安装，则能通过减缓client tick time来减轻客户端画面卡顿（前提是电脑的其他硬件没有拖累）。
